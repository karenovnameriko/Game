# Игра «Ловушка» (Avalonia)

## Описание проекта

«Ловушка» — это простая 2D-игра, разработанная с использованием фреймворка **Avalonia UI**.  
Игрок управляет персонажем-квадратом в ограниченной игровой области, избегает врагов и уничтожает их с помощью ловушек.

Размер игровой области **фиксированный** и **не изменяется при изменении размеров окна**.

---

## Игровые объекты

В игре реализованы следующие объекты:

1. **Player (Игрок)**  
   - Квадрат размером 20×20 пикселей  
   - Управляется с клавиатуры  
   - Может устанавливать ловушки  

2. **Enemy (Враг)**  
   - Красный квадрат размером 15×15 пикселей  
   - Двигается случайным образом по игровому полю  
   - Меняет направление при достижении границ поля  

3. **Trap (Ловушка)**  
   - Чёрный квадрат размером 20×20 пикселей  
   - Устанавливается игроком  
   - Уничтожает врага при столкновении  

---

## Управление

- **Стрелки клавиатуры** — перемещение игрока  
- **Пробел** — установка ловушки в текущей позиции игрока  
- **Кнопка «Начать игру»** — перезапуск игры  

Игрок не может выходить за пределы игрового поля.

---

## Игровой процесс

- Враги появляются и двигаются случайным образом  
- При столкновении врага с ловушкой:
  - враг уничтожается
  - счёт увеличивается
- При столкновении игрока с врагом — игра завершается  
- Количество врагов увеличивается с ростом счёта  
- Количество ловушек ограничено и зависит от размера игрового поля  
- Игра завершается:
  - при столкновении игрока с врагом (поражение)
  - при достижении максимального количества очков (победа)

---

## Интерфейс

- Игровое поле (Canvas фиксированного размера)
- Отображение текущего счёта
- Счётчик доступных ловушек
- Кнопка перезапуска игры
- Сообщение о победе при достижении максимального счёта

---

## Технологии

- Язык программирования: **C#**
- UI-фреймворк: **Avalonia UI**
- Платформа: **.NET**
